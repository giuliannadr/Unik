# UNIK Creative Agency 🚀

**UNIK** es una plataforma digital de vanguardia diseñada para potenciar marcas a través de experiencias visuales de alta fidelidad. Este proyecto representa la presencia digital oficial de la agencia, con un enfoque extremo en el rendimiento, la estética minimalista y la conversión de clientes.

---

## 🛠️ Stack Tecnológico

He seleccionado las herramientas más potentes del ecosistema actual para garantizar una experiencia de usuario fluida y un SEO impecable:

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/) para un tipado fuerte y seguro.
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/) con una arquitectura de diseño atómica.
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/) para micro-interacciones premium.
- **Iconografía:** [Lucide React](https://lucide.dev/) + Iconos de marca SVG optimizados.

---

## ✨ Características Principales

- **Fidelity First:** Optimización de assets para resultados "Ultra HD" y tiempos de carga mínimos.
- **Conversión Directa:** Flujos de contacto optimizados mediante integración con WhatsApp Business y Gmail.
- **Mobile Optimized:** Layouts diseñados con una jerarquía visual clara, priorizando la alineación a la izquierda en dispositivos móviles.
- **Arquitectura Limpia:** Separación estricta de componentes de UI, lógica de negocio y estilos globales.
