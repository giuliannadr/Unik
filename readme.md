# UNIK Creative Agency 🚀

**UNIK** es una plataforma digital diseñada para potenciar marcas a través de experiencias visuales de alta fidelidad. Este proyecto es el núcleo de la presencia online de la agencia, enfocado en el rendimiento, la estética minimalista y la conversión directa.

## 🛠️ Stack Tecnológico

El proyecto está construido con las herramientas más modernas del ecosistema web para garantizar velocidad y escalabilidad:

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/) para un desarrollo robusto.
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/) para un diseño responsivo y mantenible.
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/) para interacciones fluidas y orgánicas.
- **Iconografía:** [Lucide React](https://lucide.dev/) + Iconos de marca personalizados.

## ✨ Características Principales

- **Diseño Ultra HD:** Optimización de assets visuales para máxima calidad.
- **Conversión Directa:** Integración fluida con WhatsApp y Gmail para captación de leads.
- **Arquitectura de Componentes:** Código modular y reutilizable siguiendo las mejores prácticas.
- **SEO Friendly:** Optimizado para motores de búsqueda y tiempos de carga mínimos.
